If you're wondering why this site is so basic, that's because it's only just beginning. It's an experiment. An adventure. A classroom and a playground all rolled into a single repository.

One day, this site will not only be fully functional, you will be able to render it using three different frameworks (or specific lack thereof), and three different design systems:

  • React + Material UI
  • Angular + Clarity
  • Vanilla JS + Carbon

But first, it must grow from the very basics. It started with a single HTML template, and a single Javascript file. You can read more about that in this article on Medium. It will evolve, step-by-step, and each step will be explored and documented. You can follow the journey on Medium or Github.
